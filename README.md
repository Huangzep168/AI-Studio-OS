多模型 AI 项目管理中枢。统一管理栗子谷出租率系统、合同整理系统、58运营系统、商业地产获客平台、AI工作流系统、REITs研究。

## 架构

- **你 (CEO)** — 唯一决策者
- **Hermes Agent (PM + 主力开发)** — 任务拆解、分配、进度追踪、看板维护
- **Codex / Claude / DeepSeek / ChatGPT (执行层)** — 各模型负责不同任务
